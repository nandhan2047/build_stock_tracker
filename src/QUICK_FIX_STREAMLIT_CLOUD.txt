🎯 WHAT app.py DOES

```python
import sys
from pathlib import Path

root_dir = Path(__file__).parent.absolute()
sys.path.insert(0, str(root_dir))

from main import *
```

This tells Python:
1. Get the directory where app.py is (root)
2. Add that to Python's search path
3. Import everything from main.py
4. Run the Streamlit app

Simple and reliable!

❓ COMMON QUESTIONS

Q: Why do I need app.py if I have main.py?
A: Streamlit Cloud looks for entry point in root. app.py wraps main.py.

Q: Can I use main.py directly?
A: Yes, use path: build_stock_tracker/main.py

Q: Will my GitHub be messed up?
A: No, you're just adding one small file.

Q: What if it still doesn't work?
A: Check that app.py is in root, not in a subfolder.
